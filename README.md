## Getting Started
### Prerequisites

You need to install java 17 and Maven
### Installation

1Clone the repo
   ```sh
   git clone https://github.com/crudl4d/dogebook-API.git
   ```
2Install the project, either with maven or IDE
   ```sh
   mvn clean install
   ```
